# App Icon Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add the supplied sleeping-cat image as Sleepy's iOS app icon.

**Architecture:** Add one native asset catalog containing a single-size iOS 1024×1024 `AppIcon` set. Generate the PNG with a temporary Swift/Core Graphics script using disabled interpolation, then add the catalog to the existing app target's Resources phase and configure both build configurations to select it.

**Tech Stack:** Xcode 26.6, iOS 17, asset catalogs, Swift/Core Graphics, `xcodebuild`

## Global Constraints

- Preserve the supplied black background and white pixel art.
- Resize with no interpolation so pixel-art edges remain crisp.
- Do not add dependencies or modify unrelated in-progress files.
- Do not add alternate, dark, or tinted icon appearances.

- [ ] **Step 3: Resize the source image without smoothing**

Use a temporary Swift script that decodes the source, draws it into a 1024×1024 opaque RGB context with interpolation disabled, and writes a PNG to `Sleepy/Assets.xcassets/AppIcon.appiconset/AppIcon.png`:

```swift
import AppKit
import CoreGraphics
import ImageIO
import UniformTypeIdentifiers

let sourceURL = URL(fileURLWithPath: "/Users/dereksun/Downloads/images.png")
let outputURL = URL(fileURLWithPath: "Sleepy/Assets.xcassets/AppIcon.appiconset/AppIcon.png")
guard
    let source = CGImageSourceCreateWithURL(sourceURL as CFURL, nil),
    let image = CGImageSourceCreateImageAtIndex(source, 0, nil),
    let context = CGContext(
        data: nil,
        width: 1024,
        height: 1024,
        bitsPerComponent: 8,
        bytesPerRow: 0,
        space: CGColorSpaceCreateDeviceRGB(),
        bitmapInfo: CGImageAlphaInfo.noneSkipLast.rawValue
    )
else { fatalError("Unable to prepare icon") }

context.interpolationQuality = .none
context.draw(image, in: CGRect(x: 0, y: 0, width: 1024, height: 1024))
guard
    let resized = context.makeImage(),
    let destination = CGImageDestinationCreateWithURL(
        outputURL as CFURL,
        UTType.png.identifier as CFString,
        1,
        nil
    )
else { fatalError("Unable to create icon output") }

CGImageDestinationAddImage(destination, resized, nil)
guard CGImageDestinationFinalize(destination) else { fatalError("Unable to write icon") }
```

Run the script from the repository root with `swift /tmp/resize-sleepy-icon.swift`.

Expected: `Sleepy/Assets.xcassets/AppIcon.appiconset/AppIcon.png` is created.

- [ ] **Step 4: Add the catalog to the Sleepy target**

In `Sleepy.xcodeproj/project.pbxproj`:

- Add a `PBXFileReference` for `Assets.xcassets` with `lastKnownFileType = folder.assetcatalog`.
- Add a `PBXBuildFile` for `Assets.xcassets in Resources`.
- Add the file reference to the `Sleepy` group.
- Add the build file to the Sleepy target's `Resources` phase.
- Add `ASSETCATALOG_COMPILER_APPICON_NAME = AppIcon;` to the target's Debug and Release build settings.

Use unused IDs `20000000000000000000000F` for the file reference and `10000000000000000000000C` for the build file so the hand-written project file remains consistent.
